# Electric Field Visualization

A computational physics project focused on visualizing electric field behavior using Python.

This simulation models electric field interactions and field line distributions produced by charged particles. The project explores how electric fields vary spatially and provides a visual understanding of electromagnetic concepts through numerical computation.

## Concepts Explored

- Electric Fields
- Electrostatics
- Field Line Visualization
- Vector Fields
- Numerical Computation

## Tools & Libraries

- Python
- NumPy
- Matplotlib
- Jupyter Notebook

## About

Built as part of my exploration into electromagnetism, computational physics, and scientific visualization.
