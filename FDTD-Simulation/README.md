# FDTD Electromagnetic Simulation

A computational physics project implementing the Finite Difference Time Domain (FDTD) method for simulating electromagnetic wave propagation.

This simulation models the time evolution of electromagnetic waves on a discrete spatial grid using numerical techniques. The project explores wave behavior, propagation dynamics, and computational approaches commonly used in electromagnetics and physics simulations.

## Concepts Explored

- Electromagnetic Waves
- Finite Difference Time Domain (FDTD)
- Wave Propagation
- Numerical Methods
- Computational Electromagnetics

## Tools & Libraries

- Python
- NumPy
- Matplotlib
- Jupyter Notebook
