# Gravitational Slingshot Simulation

A computational simulation of the gravitational slingshot effect using Python, NumPy, and Matplotlib.

This project visualizes how a spacecraft gains velocity by passing near a massive celestial body under gravitational influence. The simulation models orbital trajectories and demonstrates the mechanics behind gravity-assisted motion used in space missions.

## Concepts Explored

- Gravitational Assist
- Orbital Mechanics
- Velocity Transfer
- Numerical Simulation
- Trajectory Visualization

## Tools & Libraries

- Python
- NumPy
- Matplotlib
- Jupyter Notebook

## Overview

The simulation numerically computes motion under gravity and visualizes the resulting trajectories in two dimensions. It was developed as part of my exploration into computational physics and scientific computing.
